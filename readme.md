# SmileX AI Prompt

AI 提示词工程仓库，用于存储和管理各类场景的提示词模板与最佳实践。

## 目录结构

```
SmileX-AI-Prompt/
├── .trae/
│   └── rules/
│       └── ai-prompt.md          # 项目约束文件
└── README.md                     # 项目说明
```

## 提示词分类

随着项目发展，将逐步包含以下类型的提示词：

- **chat** - 对话类提示词
- **generation** - 生成类提示词
- **analysis** - 分析类提示词
- **coding** - 编程辅助提示词
- **translation** - 翻译类提示词

## 快速开始

1. 浏览提示词目录，查找符合需求的提示词
2. 根据实际场景调整提示词参数
3. 在 AI 模型中测试效果
4. 如有改进建议，欢迎提交反馈

请参阅 [ai-prompt.md](./.trae/rules/ai-prompt.md) 了解提示词编写规范和版本管理要求。
