# Dictionaries - 公共 API 列表

共 **13** 个 API

### 免认证 API（5 个）

- **[Chinese Character Web](http://ccdb.hemiola.com/)** - Chinese character definitions and pronunciations | HTTPS: No | CORS: No
- **[Chinese Text Project](https://ctext.org/tools/api)** - Online open-access digital library for pre-modern Chinese texts | HTTPS: Yes | CORS: Unknown
- **[Free Dictionary](https://dictionaryapi.dev/)** - Definitions, phonetics, pronounciations, parts of speech, examples, synonyms | HTTPS: Yes | CORS: Unknown
- **[Indonesia Dictionary](https://new-kbbi-api.herokuapp.com/)** - Indonesia dictionary many words | HTTPS: Yes | CORS: Unknown
- **[Wiktionary](https://en.wiktionary.org/w/api.php)** - Collaborative dictionary data | HTTPS: Yes | CORS: Yes

### 需认证 API（8 个）

- **[Collins](https://api.collinsdictionary.com/api/v1/documentation/html/)** - Bilingual Dictionary and Thesaurus Data | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Lingua Robot](https://www.linguarobot.io)** - Word definitions, pronunciations, synonyms, antonyms and others | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
- **[Merriam-Webster](https://dictionaryapi.com/)** - Dictionary and Thesaurus Data | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[OwlBot](https://owlbot.info/)** - Definitions with example sentence and photo if available | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
- **[Oxford](https://developer.oxforddictionaries.com/)** - Dictionary Data | 认证: `apiKey` | HTTPS: Yes | CORS: No
- **[Synonyms](https://www.synonyms.com/synonyms_api.php)** - Synonyms, thesaurus and antonyms information for any given word | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Wordnik](https://developer.wordnik.com)** - Dictionary Data | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Words](https://www.wordsapi.com/docs/)** - Definitions and synonyms for more than 150,000 words | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
