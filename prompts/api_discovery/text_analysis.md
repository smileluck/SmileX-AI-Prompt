# Text Analysis - 公共 API 列表

共 **15** 个 API

### 免认证 API（1 个）

- **[LibreTranslate](https://libretranslate.com/docs)** - Translation tool with 17 available languages | HTTPS: Yes | CORS: Unknown

### 需认证 API（14 个）

- **[Code Detection API](https://codedetectionapi.runtime.dev)** - Detect, label, format and enrich the code in your app or in your data pipeline | 认证: `OAuth` | HTTPS: Yes | CORS: Unknown
- **[apilayer languagelayer](https://languagelayer.com/)** - Language Detection JSON API supporting 173 languages | 认证: `OAuth` | HTTPS: Yes | CORS: Unknown
- **[Aylien Text Analysis](https://docs.aylien.com/textapi/#getting-started)** - A collection of information retrieval and natural language APIs | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Cloudmersive Natural Language Processing](https://www.cloudmersive.com/nlp-api)** - Natural language processing and text analysis | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
- **[Detect Language](https://detectlanguage.com/)** - Detects text language | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[ELI](https://nlp.insightera.co.th/docs/v1.0)** - Natural Language Processing Tools for Thai Language | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Google Cloud Natural](https://cloud.google.com/natural-language/docs/)** - Natural language understanding technology, including sentiment, entity and syntax analysis | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Hirak OCR](https://ocr.hirak.site/)** - Image to text -text recognition- from image more than 100 language, accurate, unlimited requests | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Hirak Translation](https://translate.hirak.site/)** - Translate between 21 of most used languages, accurate, unlimited requests | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Lecto Translation](https://rapidapi.com/lecto-lecto-default/api/lecto-translation/)** - Translation API with free tier and reasonable prices | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
- **[Semantria](https://semantria.readme.io/docs)** - Text Analytics with sentiment analysis, categorization & named entity extraction | 认证: `OAuth` | HTTPS: Yes | CORS: Unknown
- **[Sentiment Analysis](https://www.meaningcloud.com/developer/sentiment-analysis)** - Multilingual sentiment analysis of texts from different sources | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
- **[Tisane](https://tisane.ai/)** - Text Analytics with focus on detection of abusive content and law enforcement applications | 认证: `OAuth` | HTTPS: Yes | CORS: Yes
- **[Watson Natural Language Understanding](https://cloud.ibm.com/apidocs/natural-language-understanding/natural-language-understanding)** - Natural language processing for advanced text analysis | 认证: `OAuth` | HTTPS: Yes | CORS: Unknown
