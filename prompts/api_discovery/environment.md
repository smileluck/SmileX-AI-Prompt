- **[National Grid ESO](https://data.nationalgrideso.com/)** - Open data from Great Britain’s Electricity System Operator | HTTPS: Yes | CORS: Unknown
- **[PM2.5 Open Data Portal](https://pm25.lass-net.org/#apis)** - Open low-cost PM2.5 sensor data | HTTPS: Yes | CORS: Unknown
- **[UK Carbon Intensity](https://carbon-intensity.github.io/api-definitions/#carbon-intensity-api-v1-0-0)** - The Official Carbon Intensity API for Great Britain developed by National Grid | HTTPS: Yes | CORS: Unknown
- **[Website Carbon](https://api.websitecarbon.com/)** - API to estimate the carbon footprint of loading web pages | HTTPS: Yes | CORS: Unknown
