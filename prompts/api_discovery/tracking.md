# Tracking - 公共 API 列表

共 **9** 个 API

### 免认证 API（3 个）

- **[PostalPinCode](http://www.postalpincode.in/Api-Details)** - API for getting Pincode details in India | HTTPS: Yes | CORS: Unknown
- **[Postmon](http://postmon.com.br)** - An API to query Brazilian ZIP codes and orders easily, quickly and free | HTTPS: No | CORS: Unknown
- **[WhatPulse](https://developer.whatpulse.org/#web-api)** - Small application that measures your keyboard/mouse usage | HTTPS: Yes | CORS: Unknown

### 需认证 API（6 个）

- **[Aftership](https://developers.aftership.com/reference/quick-start)** - API to update, manage and track shipment efficiently | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
- **[Correios](https://cws.correios.com.br/ajuda)** - Integration to provide information and prepare shipments using Correio's services | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Pixela](https://pixe.la)** - API for recording and tracking habits or effort, routines | 认证: `X-Mashape-Key` | HTTPS: Yes | CORS: Yes
- **[PostNord](https://developer.postnord.com/api)** - Provides information about parcels in transport for Sweden and Denmark | 认证: `apiKey` | HTTPS: No | CORS: Unknown
- **[UPS](https://www.ups.com/upsdeveloperkit)** - Shipment and Address information | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[WeCanTrack](https://docs.wecantrack.com)** - Automatically place subids in affiliate links to attribute affiliate conversions to click data | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
