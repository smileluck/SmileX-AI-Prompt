# Patent - 公共 API 列表

共 **4** 个 API

### 免认证 API（2 个）

- **[PatentsView ](https://patentsview.org/apis/purpose)** - API is intended to explore and visualize trends/patterns across the US innovation landscape | HTTPS: Yes | CORS: Unknown
- **[USPTO](https://www.uspto.gov/learning-and-resources/open-data-and-mobility)** - USA patent api services | HTTPS: Yes | CORS: Unknown

### 需认证 API（2 个）

- **[EPO](https://developers.epo.org/)** - European patent search system api | 认证: `OAuth` | HTTPS: Yes | CORS: Unknown
- **[TIPO](https://tiponet.tipo.gov.tw/Gazette/OpenData/OD/OD05.aspx?QryDS=API00)** - Taiwan patent search system api | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
