# Data Validation - 公共 API 列表

共 **7** 个 API

### 免认证 API（2 个）

- **[Postman Echo](https://www.postman-echo.com)** - Test api server to receive and return value from HTTP method | HTTPS: Yes | CORS: Unknown

- **[US Extract](https://www.smarty.com/products/apis/us-extract-api)** - Extract postal addresses from any text including emails | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
- **[US Street Address](https://www.smarty.com/docs/cloud/us-street-api)** - Validate and append data for any US postal address | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
