# Calendar - 公共 API 列表

共 **16** 个 API

### 免认证 API（10 个）

- **[Church Calendar](http://calapi.inadiutorium.cz/)** - Catholic liturgical calendar | HTTPS: No | CORS: Unknown
- **[Czech Namedays Calendar](https://svatky.adresa.info)** - Lookup for a name and returns nameday date | HTTPS: No | CORS: Unknown
- **[Hebrew Calendar](https://www.hebcal.com/home/developer-apis)** - Convert between Gregorian and Hebrew, fetch Shabbat and Holiday times, etc | HTTPS: No | CORS: Unknown
- **[LectServe](http://www.lectserve.com)** - Protestant liturgical calendar | HTTPS: No | CORS: Unknown
- **[Nager.Date](https://date.nager.at)** - Public holidays for more than 90 countries | HTTPS: Yes | CORS: No
- **[Namedays Calendar](https://nameday.abalin.net)** - Provides namedays for multiple countries | HTTPS: Yes | CORS: Yes
- **[Non-Working Days](https://github.com/gadael/icsdb)** - Database of ICS files for non working days | HTTPS: Yes | CORS: Unknown
- **[Non-Working Days](https://isdayoff.ru)** - Simple REST API for checking working, non-working or short days for Russia, CIS, USA and other | HTTPS: Yes | CORS: Yes
- **[Russian Calendar](https://github.com/egno/work-calendar)** - Check if a date is a Russian holiday or not | HTTPS: Yes | CORS: No
- **[UK Bank Holidays](https://www.gov.uk/bank-holidays.json)** - Bank holidays in England and Wales, Scotland and Northern Ireland | HTTPS: Yes | CORS: Unknown

### 需认证 API（6 个）

- **[Calendarific](https://calendarific.com/)** - Worldwide Holidays | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Checkiday - National Holiday API](https://apilayer.com/marketplace/checkiday-api)** - Industry-leading Holiday API. Over 5,000 holidays and thousands of descriptions. Trusted by the World’s leading companies | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Festivo Public Holidays](https://docs.getfestivo.com/docs/products/public-holidays-api/intro)** - Fastest and most advanced public holiday and observance service on the market | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
- **[Google Calendar](https://developers.google.com/google-apps/calendar/)** - Display, create and modify Google calendar events | 认证: `OAuth` | HTTPS: Yes | CORS: Unknown
- **[Holidays](https://holidayapi.com/)** - Historical data regarding holidays | 认证: `apiKey` | HTTPS: Yes | CORS: Unknown
- **[Public Holidays](https://www.abstractapi.com/holidays-api)** - Data on national, regional, and religious holidays via API | 认证: `apiKey` | HTTPS: Yes | CORS: Yes
