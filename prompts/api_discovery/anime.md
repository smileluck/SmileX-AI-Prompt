- **[Jikan](https://jikan.moe)** - Unofficial MyAnimeList API | HTTPS: Yes | CORS: Yes
- **[NekosBest](https://docs.nekos.best)** - Neko Images & Anime roleplaying GIFs | HTTPS: Yes | CORS: Yes
